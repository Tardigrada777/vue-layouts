#### vue-layouts (l5s)

Small, simple and fast lib for convinience building layouts with Vue 3.

#### Warning!

Currently is under hard development. If you have any ideas how to improve this lib,
PR's are welcome.

#### development and contributing

1. **install deps**

```bash
yarn
```

2. **run dev server**

```bash
yarn serve
```
